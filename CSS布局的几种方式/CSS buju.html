使用max-width来使窗口与盒模型宽度自适应

用max-width来替代width可以使浏览器更好的处理小窗口的情况，这点在移动设备上显得很重要。设置为max-width当窗口变小的时候，文章内容会随着窗口的变化自动伸缩，设置为width，窗口变小时，内容也会随着窗口变小隐藏。

使用box-sizing使padding和border不增加视觉宽度

如果设置盒子模型的宽度是一样的，但是由于border和padding属性会使盒模型的视觉宽度不同。此时，用border-sizing：border-box；声明可以使内边距padding和边框border不会增加他的宽度。
